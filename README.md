# project Q.Q
**[ qook & qlick ]** <br> 쿡앤클릭 : 회원가입부터 메뉴주문까지 클릭 몇번으로 가능합니다!<br><br>

## 프로젝트 소개
식당에서의 간편한 주문과 운영을 위한 비대면 음식 주문 서비스
- Admin : 일별, 월별 매출 / 회원 주문내역 조회, 메뉴판 관리 등 매장 운영 편의 기능 제공
- Customer : 메뉴 열람, 장바구니 / 주문 / 결제 기능, 본인 주문 내역 확인<br><br>
